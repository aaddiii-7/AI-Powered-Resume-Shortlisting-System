# 🚀 Resumind – AI Resume Analyzer & ATS Scoring System

---

## 📌 Introduction

Resumind is an AI-powered resume analysis platform designed to help users evaluate their resumes against job descriptions and improve their chances of passing Applicant Tracking Systems (ATS).

The application allows users to upload resumes, analyze them using AI, and receive structured feedback including ATS scores, keyword gaps, and improvement suggestions.


---

## ⚙️ Tech Stack

* **React**
  React is a popular open-source JavaScript library for building user interfaces using reusable components and a virtual DOM, enabling efficient, dynamic single-page and native apps.

* **React Router v7**
  React Router v7 is the go-to routing library for React apps, offering nested routes, data loaders/actions, error boundaries, code splitting, and SSR support—all with a smooth upgrade path from v6.

* **Puter.js**
  Puter.js is a lightweight client-side SDK that provides serverless authentication, storage, database, and AI capabilities directly in the browser—no backend required.

* **Tailwind CSS**
  Tailwind CSS is a utility-first CSS framework that enables rapid UI development using low-level utility classes.

* **TypeScript**
  TypeScript is a superset of JavaScript that adds static typing, improving code quality, maintainability, and developer experience.

* **Vite**
  Vite is a fast build tool and development server with instant startup, hot module replacement, and optimized production builds.

* **Zustand**
  Zustand is a minimal, hook-based state management library for React that allows efficient global state management with zero boilerplate.

## 🖼️ Application Workflow & Usage

### 🔄 How the App Works

1. User logs in using browser-based authentication
2. Uploads a resume (PDF/Text)
3. (Optional) Adds a job description
4. AI processes the resume
5. System generates:

   * ATS Score
   * Resume-job match analysis
   * Missing keywords
   * Improvement suggestions
